Heartbeat monitoring

Ping an uptime monitor after each scheduled run — it alerts you if this updater stops, in case of app crash and notification can't be sent. Supports Healthchecks.io, Uptime Kuma, Better Stack, and a Custom URL.

Backup & restore

Download your whole configuration as a file, or restore one onto this instance — handy for moving between servers. Copying /data/config.json directly does the same thing and never exposes secrets to your browser.

Download backup

⚠ The file contains your API tokens and passwords in plaintext. Store it somewhere safe.

Restore from backup

⚠ This overwrites the entire configuration on this instance — zones, tokens, WAF lists, notifications, everything. It cannot be undone.
